# FSETC Evaluation API

Spring Boot Server 


## Overview 
This API was generated by swagger-codegen and implemented by Steven Murray, Cole Stanfield, and Jedde Bowman.

To run locally, start as a simple java project and connect to http://localhost:8080 to view the Swagger documentation and test endpoints.

## Installing to server
1. Build a war by running the maven target
    ```shell script
    mvn clean package
    ```
2. Copy the war to the server under your home directory
3. Login to server and as root move the war file to /home/tomcat/webapps
4. Run the shutdown script located in /home/tomcat/bin then restart with catalina script
    ```shell script
    ./bin/shutdown.sh
    ./bin/catalina.sh run
    ```
